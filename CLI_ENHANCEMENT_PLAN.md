## Implemented Enhancements

### 1. Enhanced `validate_catalog` CLI

**Enhancement**: Added `--summary` flag for machine-readable output and improved quality check flags.

```bash
# Basic validation (quality checks enabled by default)
# Note: Quality checks require the 'quality' extra: pip install imas-standard-names[quality]
validate_catalog standard_names/ --summary text
# Output: ✓ Validated 305 entries (0 errors, 134 warnings)

# Skip quality checks for faster validation (or if 'quality' extra not installed)
validate_catalog standard_names/ --no-quality-check --summary text

# Strict mode: fail on warnings
validate_catalog standard_names/ --strict --summary text

# JSON output
validate_catalog standard_names/ --summary json
# Output: {"passed": true, "entries": 305, "errors": 0, "warnings": 134, ...}
```

### 2. Enhanced `standard-names build` CLI

**Enhancement**: Added `--verify` flag for verification output.

```bash
standard-names build standard_names/ --db catalog.db --verify
# Output: ✓ Built catalog.db: 45.2 KB, 305 entries
```

### 3. Added `standard-names docs` CLI

**Purpose**: Build documentation for catalog using mkdocs + mike.

```bash
standard-names docs build <catalog_path> --version VERSION [--push] [--set-default]
standard-names docs alias --version VERSION --alias ALIAS [--push]
```

### Architecture

The catalog documentation is now generated by the catalog repository using:

1. `CatalogRenderer` class for rendering catalog as Markdown
2. `standard-names docs build` command for building mkdocs site
3. `mike` for versioned documentation deployment

This allows the catalog repository to remain zero-code while getting rich documentation.

## CI Workflow Example

```yaml
name: Catalog CI

on:
  push:
    branches: [main]
  pull_request:

jobs:
  validate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Install uv
        uses: astral-sh/setup-uv@v4
      
      - name: Install dependencies
        run: uv pip install imas-standard-names
      
      - name: Validate catalog
        run: uv run validate_catalog standard_names/ --summary text
      
      - name: Build catalog database
        run: uv run standard-names build standard_names/ --db catalog.db --verify

  docs:
    runs-on: ubuntu-latest
    needs: validate
    if: github.ref == 'refs/heads/main'
    steps:
      - uses: actions/checkout@v4
      
      - name: Install dependencies
        run: |
          uv pip install imas-standard-names
          uv pip install mike mkdocs-material
      
      - name: Deploy docs
        run: |
          uv run standard-names docs build standard_names/ \
            --version \${{ github.ref_name }} \
            --push \
            --set-default
```
